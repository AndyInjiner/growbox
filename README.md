# 🌱 ESP32 Growbox System

Система управления гроубоксом на базе ESP32 с симуляцией в Wokwi.

[![Wokwi](https://img.shields.io/badge/Wokwi-Simulate-blue)](https://wokwi.com/github/Andyinjiner/Growbox)

## Возможности

- **7 режимов работы:**
  1. Вентиляция по влажности/температуре
  2. Вентиляция периодическая
  3. Освещение (восход/закат)
  4. Насос наружный по влажности
  5. Насос наружный периодический
  6. Насос почвы по влажности
  7. Обогрев по температуре

- **TFT дисплей** с меню управления
- **Web-интерфейс** для удалённого управления
- **Датчики:** температура, влажность воздуха, влажность почвы
- **Сохранение настроек** в энергонезависимой памяти

## Запуск симуляции

1. Откройте репозиторий на GitHub
2. Нажмите на бейдж **Wokwi Simulate**
3. Или перейдите по ссылке: [https://wokwi.com/github/Andyinjiner/Growbox](https://wokwi.com/github/Andyinjiner/Growbox)

## Подключение

| Компонент | Пин ESP32 |
|-----------|-----------|
| DHT22 (внутри) | GPIO 32 |
| DHT22 (снаружи) | GPIO 33 |
| Soil Moisture | GPIO 34 |
| Кнопка MODE | GPIO 14 |
| Кнопка CONFIRM | GPIO 15 |
| Вентиляция | GPIO 2 |
| Свет (PWM) | GPIO 12 |
| Насос (наружный) | GPIO 4 |
| Насос (почва) | GPIO 13 |
| Обогрев | GPIO 25 |
| Зуммер | GPIO 26 |
| TFT Display | SPI (GPIO 5, 16, 17, 18, 19, 23) |

## Web-интерфейс

После запуска подключитесь к:
- **AP Mode:** `Grobox_AP` / `grobox123`
- **WiFi Mode:** IP адрес в Serial мониторе

Откройте в браузере: `http://192.168.4.1` или `http://<IP_устройства>`
