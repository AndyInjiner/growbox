# 🎮 Симуляция Wokwi для проекта Growbox

Этот файл содержит схему для симуляции проекта в онлайн-симуляторе [Wokwi](https://wokwi.com).

## 🚀 Быстрый запуск

1. Откройте [Wokwi ESP32 Simulator](https://wokwi.com/esp32)
2. Нажмите **"Import from GitHub"**
3. Введите URL репозитория: `https://github.com/Andyinjiner/Growbox`
4. Нажмите **Import**

Или откройте прямую ссылку (после загрузки файлов в репозиторий):
```
https://wokwi.com/projects/github/Andyinjiner/Growbox
```

## 📋 Схема подключений

### Датчики
| Компонент | GPIO | Питание |
|-----------|------|---------|
| DHT22 (внутри) | 32 | 3.3V |
| DHT22 (снаружи) | 33 | 3.3V |
| Soil Moisture | 34 (ADC) | 3.3V |

### Исполнительные устройства
| Устройство | GPIO | Тип |
|------------|------|-----|
| Реле вентиляции | 2 | 5V реле |
| Реле насоса (out) | 4 | 5V реле |
| Реле насоса (soil) | 13 | 5V реле |
| Реле обогрева | 25 | 5V реле |
| MOSFET (Light PWM) | 12 | 5V |
| Зуммер | 26 | 5V (через транзистор) |

### Интерфейс
| Компонент | GPIO |
|-----------|------|
| TFT CS | 5 |
| TFT DC | 16 |
| TFT RST | 17 |
| TFT MOSI | 23 |
| TFT SCK | 18 |
| TFT MISO | 19 |
| Кнопка MODE | 14 |
| Кнопка CONFIRM | 15 |

##  Как использовать симуляцию

1. **Просмотр датчиков**: Нажмите кнопку MODE для просмотра показаний
2. **Выбор режима**: Длительное нажатие MODE для выбора режима
3. **Активация режима**: Нажмите CONFIRM для активации
4. **Web интерфейс**: Откройте http://192.168.4.1 в браузере симулятора

## ⚠️ Ограничения симуляции

- Wokwi не симулирует WiFi полностью (только логирование)
- Датчики DHT22 возвращают фиксированные значения
- Аналоговый датчик почвы требует ручной настройки в симуляции
- TFT дисплей может отображаться некорректно (зависит от библиотеки)
